## 📌 Project Overview
**Hope Hospital** is a desktop-based healthcare management solution developed for the **Object-Oriented Programming 2 (OOP2)** course. The application provides an integrated platform for managing patient appointments, doctor schedules, and administrative tasks, ensuring a streamlined workflow for healthcare providers and a user-friendly experience for patients.

## 🚀 Key Features
- **Secure Authentication:** Multi-level login system for Admins and Patients, with backend validation via `dblogin.php` logic (integrated via JDBC).
- **Advanced Appointment Booking:** Users can browse doctors, check real-time availability, and book slots in specific clinics (Dental, etc.).
- **Staff & Services Directory:** A dynamic interface to view specialized medical departments and available doctors.
- **Data Persistence:** Full database integration using **MySQL**, handling appointments, doctor lists, and contact inquiries.
- **Communication Module:** A dedicated "Contact Us" form allowing patients to send inquiries directly to the hospital administration.

## 🛠️ Tech Stack
- **Language:** Java (JDK 8+).
- **UI Framework:** Java Swing (NetBeans GUI Builder).
- **Backend Logic:** OOP principles (Encapsulation, Inheritance).
- **Database Connection:** JDBC (Java Database Connectivity).
- **Database Engine:** MySQL.

## 📁 Repository Structure
- `HomePage.java`: The main navigation dashboard.
- `Admin.java`: Administrative control and secure login handling.
- `Book.java`: Core logic for searching and reserving medical appointments.
- `Doctors.java`: Directory of healthcare professionals and their schedules.
- `Contact.java`: Inquiry management system for user feedback.
- `About.java`: Presentation of the hospital's vision and mission.

## ⚙️ Database Setup
The system connects to a local MySQL instance. To set up the environment:
- **Database Name:** `hope`
- **Connection String:** `jdbc:mysql://localhost:3306/hope`
- **Credentials:** Default (User: `root`, Password: `1234`).
